# Combine pdf files

*September 18, 2018*

Sometimes you have multiple PDF files you want to put together into one file.

If you have linux, it couldn’t be easier using the convert command.

|  |  |
| --- | --- |
| You have to have the files in the same folder. This will take file1 and append it to file2 and save the output as fileOut.pdf. | Covert file1.pdf file2.pdf fileOut.pdf |
| Practical Example: | convert 1-4.pdf vwbuybackacceptance.pdf 6-7.pdf vwnotarized.pdf VWFinal.pdf |
