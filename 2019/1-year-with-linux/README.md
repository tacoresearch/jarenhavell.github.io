| About a year ago, I started playing around with linux as my primary operating system. I had played with linux many times, loading it up on old computers or using it in specific instances at work for items like syslog and email filtering. I had a working knowledge, but it seemed like every time I tried to do simple things it was a challenge because it would be months between sessions. I decided that I wanted to play with it more, so I began trying out differnt linux flavors on an old laptop. I explored distros such as ubuntu, lubuntu, fedora, and finally Debian. I settled on Debian as I am not particularly fond of some of the ways Canonical seems to be trying to monetize ubuntu, but at the same time it is closely coupled with ubuntu’s development. Additionally, I thought ubuntu would have been “too easy”, and I wanted to challenge myself with something new and unfamiliar. |

Boy was I in for a surprise.

A few highlights of what I learned:

- You have to [install Sudo](http://jarenhavell.com/2018/02/04/install-sudo-on-debian/).
- One does not simply [apt-get install chrome](http://jarenhavell.com/2018/02/07/install-chrome-on-debian/).
- [Cloning and resizing an ext4 partition](http://www.jarenhavell.com/2018/12/26/cloning-a-debian-system/) is pretty easy.
- [Screenfetch and Neofetch](http://jarenhavell.com/2018/02/22/screenfetch-and-neofetch/) are handy utilities.
- [DD for working](http://jarenhavell.com/2018/05/04/dd-writing-an-img-or-iso-to-a-disk/) with images and iso files.
- Combining PDFs is [wicked easy](http://jarenhavell.com/2018/09/18/combine-pdf-files/).

I would encourage you to give it a try. It took me acquiring a “[new](https://www.ebay.com/sch/i.html?_from=R40&_trksid=m570.l1313&_nkw=thinkpad+i5+8gb+ssd&_sacat=0&LH_TitleDesc=0&_sop=15&_osacat=0&_odkw=thinkpad+i5+ssd&LH_BIN=1)” dedicated machine for me to commit to this, and I forced myself to use it 95% of the time. I still occasionally need use my other machines (Windows and mac) when I have tasks that “just need to get done”.

Cheers to another great year of learning!
